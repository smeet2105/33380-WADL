FaceTrace

The aim of this project was to identify whether a photograph is deemed fit to be set as a profile picture for professional front. The models were trained to detect whether the image was blurry, whether more than one faces are detected, whether mask or something covers the face, or if any animals are there in the picture. FaceTrace was built using ReactJs, Flask, TensorFlow, Keras.
